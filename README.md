Masaüstü Personel Yönetim Sistemi geliştirdim. WinForms tabanlı uygulamada kullanıcı giriş/kayıt, personel ekleme–güncelleme–silme (soft delete), arama/filtreleme, maaş yönetimi ve dashboard istatistikleri (toplam/aktif/pasif, son eklenenler) gibi modüller tasarladım. Verileri ADO.NET ile SQL Server veritabanında yönettim ve arayüzde tema/renk bileşenlerini standartlaştırdım.

Öne Çıkanlar

CRUD işlemleri ve soft delete mantığı
Dashboard metrikleri ve son eklenen çalışan listesi
Maaş güncelleme iş akışı
Kullanıcı doğrulama ve kayıt
UI tema yönetimi ve grid stilleri


Teknolojiler

C#, WinForms, .NET Framework 4.7.2
ADO.NET (SqlConnection/SqlCommand/SqlDataReader)
SQL Server / LocalDB
DataGridView, Windows Forms UI


Developed a desktop Employee Management System. The WinForms app includes user login/registration, employee add–update–soft delete, search/filtering, salary management, and dashboard analytics (total/active/inactive, recent hires). Data is managed in SQL Server via ADO.NET, and the UI uses a consistent theme/styling layer.

Highlights

CRUD operations with soft delete
Dashboard metrics and recent employees list
Salary update workflow
User authentication and registration
Centralized UI theming and DataGridView styling
